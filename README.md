# 🎮 ¿Quién es este personaje de Disney? – Drag & Drop Game (HTML · CSS · JavaScript)

![Banner Juego Disney](https://github.com/mariarosete/disney-drag-drop/blob/main/screenshots/disney.png?raw=true)

Aplicación web interactiva desarrollada con JavaScript que implementa funcionalidades de Drag & Drop, validación visual y manipulación dinámica del DOM.

## 🚀 Funcionalidades

- Drag & Drop con la API nativa de JavaScript  
- Validación visual de aciertos y errores  
- Generación aleatoria de personajes en cada partida  
- Sistema de victoria con feedback visual  
- Carga dinámica de datos desde JSON  

---

## 🧠 Aspectos técnicos

- Uso de eventos: `dragstart`, `dragover`, `drop`  
- Manipulación del DOM  
- Fetch de datos desde JSON  
- Lógica de validación y estado del juego  
---

## 🎯 Objetivo

- Practicar **JavaScript Vanilla**  
- Implementar la **Drag & Drop API**  
- Trabajar lógica de interacción y validación  
- Mejorar la experiencia de usuario
